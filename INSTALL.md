# Classroom Slide Builder — Install Guide

## One-time setup (takes ~1 minute)

1. **Download the plugin files**
   - Go to https://github.com/BRANDONIBEN/classroom-slide-builder
   - Click the green **Code** button > **Download ZIP**
   - Unzip to a permanent location (e.g., `Documents/classroom-slide-builder/`)

2. **Load in Figma**
   - Open Figma
   - Go to **Plugins** menu (top bar) > **Development** > **Import plugin from manifest...**
   - Navigate to the folder you unzipped and select `manifest.json`
   - The plugin now appears under **Plugins > Development > Classroom Slide Builder**

3. **Run the plugin**
   - In any Figma file: **Plugins** > **Development** > **Classroom Slide Builder**

## Updating the plugin

When Brandon pushes an update:
1. Re-download the ZIP from GitHub (or `git pull` if you cloned)
2. Replace the old files — Figma reads from your local folder, so it updates automatically
